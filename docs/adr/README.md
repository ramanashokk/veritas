# Architectural Decision Records

This directory contains Architectural Decision Records (ADRs).

Each ADR documents a significant architectural decision, including:

- Context
- Decision
- Alternatives Considered
- Consequences

Naming convention:

ADR-001-provider-sdk.md
ADR-002-consensus-engine.md
ADR-003-verification-engine.md

ADRs are immutable historical records.

If a decision changes, create a new ADR rather than editing the old one.
