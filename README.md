# Veritas

> **The Evidence Engine**

Veritas is an open platform that helps people understand what the scientific evidence actually says.

Instead of relying on a single study or AI-generated opinion, Veritas analyzes published research, evaluates evidence quality, and builds transparent scientific consensus.

**AI is not the authority. The evidence is.**

## Vision

To become the world's most trusted platform for understanding scientific evidence.

## Principles

- Evidence over opinion
- Transparency by default
- Every conclusion is traceable
- Uncertainty is shown, not hidden
- Science evolves, so Veritas evolves

## Status

🚧 Early development

## Roadmap

- [ ] Ask scientific questions
- [ ] Retrieve relevant research
- [ ] Rank evidence quality
- [ ] Build scientific consensus
- [ ] Explain findings in plain English
- [ ] Cite every source
